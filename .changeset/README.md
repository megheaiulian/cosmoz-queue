# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`.

A changeset is a way to describe the changes you're making to your codebase. It describes a change that you want to make to your project and adds it to the changelog.

## When to add a changeset

- When you're adding a new feature
- When you're fixing a bug
- When you're improving performance
- When you're making documentation changes
- When you're changing the public API

## How to add a changeset

1. Run `npm run changeset`
2. Select the packages you've changed
3. Select the type of change (patch, minor, or major)
4. Write a summary of the change
5. Commit the changeset file

For more information, see the [changesets documentation](https://github.com/changesets/changesets).
